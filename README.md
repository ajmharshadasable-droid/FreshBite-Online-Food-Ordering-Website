# 🍔 FreshBite – Online Food Ordering Website

## 📖 Project Description

FreshBite is a responsive online food ordering website developed using HTML, CSS, and JavaScript. Users can browse food items, manage their cart, apply coupons, choose delivery times, and place orders through a simple and user-friendly interface.

## ✨ Features

- 🏠 Home Page
- 🍕 Food Menu
- 🍽️ Restaurants Page
- 🛒 Shopping Cart
- 💳 Checkout Page
- 🎟️ Coupon Code
- 🚚 Delivery Time Selection
- ❤️ Wishlist
- 👤 Login Page
- ℹ️ About Page
- 📞 Contact Page
- 📱 Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage
- Font Awesome
